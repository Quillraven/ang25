A long, long time ago, when I was starting game development in the Windows Console with C++, I had an idea for a game called Masamune. You can find this first version on YouTube.

Last year (2024) I picked up that idea again and now implemented it in LibGDX using Kotlin. All the ideas (and more!) I had in mind are finalized.

You can play the beginning of the story, where you learn a little bit about the fantasy world you are in and about the legendary sword Masamune!

It is nothing impressive. So don't expect a lot. The play time is ~30 minutes I guess.

This version contains:

  • A controls screen
  • A main menu screen for music/sound volume, language settings, and save state loading
  • An introduction cut scene that can be canceled
  • Four different maps that you can discover
  • Multiple quests, monsters, and items
  • A main quest cut scene with a Boss fight
  • An outro sequence when this "demo" is over

The majority of assets are from itch.io. Any donations towards Masamune will also go back to the assets creators. They are all mentioned in the credits on the GitHub page.

I appreciate any feedback and thanks for playing! :)
